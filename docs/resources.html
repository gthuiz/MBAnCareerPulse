Resources

Becoming a data professional is complicated in that you are expected to understand or have experience with many forms of technology. This section will detail many of the items you’ll be expected to know, and provide helpful resources to learn them if not.

Python

Summary:

Python is a very strong and useful data collection and visualization tool in today’s big data world. With Python, we can collect data from many websites and other files in a flash and analyze data to reach conclusions in various ways. Now, Python is widely used in many industries such as Technology, Business, and Arts. Also, people who use Python fluently are likely to have higher salaries than those without such skill.

Customers:

Python website

Ways to learn:

  • Practice by downloading Python 3.x and VSCode according to your computer system.
  • Download Python Cheat Sheet in your computer for looking up.
  • Great YouTube Tutorial for beginners by Learn Python - Full Course for Beginners and for deep learning by Python Full Course
  • Free Coursera for Umich: Python 3 Programming and Statistics with Python
  • To do further data valiation and statistics, Pandas Library and Numpy Library maybe helpful for you.
  • For Umich Ross MBAn Students, you could have Python related courses in MBAN 550:Introduction to Data Programming. Click on the link to learn about the curriculum.

MySQL

Summary:

MySQL is an open-source relational database management system. Its name is a combination of “My”, the name of co-founder Michael Widenius’s daughter My, and “SQL”, the abbreviation for Structured Query Language.

Customers:

BBC, facebook, NASA, Symantec, JP Morgan, Verizon Full List MySQL Website

Ways to Learn:

  • Practice by downloading MySQL Desktop
  • Great YouTube Tutorial for beginners by Programming with Mosh(This guy is all you need to start!!)
  • Another MySQL Tutorial website if you don’t like YouTube: MySQLTutorial
  • Recommended Coursera’s course: Learn SQL Basics for Data Science Specialization from UC Davis
  • Free Coursera course from Umich: Introduction to Structured Query Language (SQL)
  • Datacamp is another great website to study and pratice SQL

Business Intelligence Tools

Business Intelligence (BI) Tools are types of software that are used to process and display data. Actionable insights and key decisions come out of using these BI systems, and their impact on business operations is continuing to grow.

Below are details on a few of the most popular ones:

Qlik Sense

Summary:

Qlik’s suite of analytics tools allows customers to create visualizations and automated reports to help unlock insights from their data. Qlik was founded in 1993 and is a well known players in the analytics field.

Customers:

PayPal, BP, Deloitte, Urban Outfitters, Airbus, Volvo, Ford

Full List Qlik Sense Website

Ways to Learn:

  • Practice by downloading Qlik Sense Desktop
  • Utilize free data sets at kaggle to build your dashboards
  • Qlik Sense training courses are offered on LinkedIn Learning
  • Qlik Learning Portal
  • Popular YouTube channel: Abhishek Agarrwal

Power BI

Summary:

Created by Microsoft in 2011, Power BU is a suite of BI tools that enable customers to develop reports, visualizations, and dashboards. “Power BI,” Microsoft says, “is a business analytics solution that lets you visualize your data and share insights across your organization, or embed them in your app or website.” It integrates well with Microsoft’s suite of products and services, and is the logical BI choice for companies that utilize other Microsoft products.

Customers:

Hewlett-Packard, KPMG, EY, PwC, Nestle, Rolls Royce

  • Utilize free data sets at kaggle to build your dashboards
  • Power BI training courses are offered on LinkedIn Learning
  • Complete Introduction course on Udemy
  • Popular YouTube channels: edureka!, Avi Singh - PowerBIPro, Kevin Stratvert

Tablaeu

Summary:

Tableau data visualization software helps explore relational databases and data cubes through dashboards and reports. Tablaeu was acquired by Salesforce in 2019 for $15.7 billion, and its technology and software is now utilized on the largest CRM platform.

Customers:

Nike, Coca-Cola, Skype, The World Bank, Wells Fargo, Citigroup, Amica, The New York Times
